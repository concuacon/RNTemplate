## :arrow_forward: How to Run App
```
1. cd to the repo
2. Run Build for either OS
  * for iOS
    * run `npm run i` or `react-native run-ios`
  * for Android
    * Run Genymotion
    * run  `npm run a` or `react-native run-android
```
